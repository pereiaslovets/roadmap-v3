1.1. Вывести к-во всех постов.

		SELECT COUNT(*) FROM post
		
1.2. Вывести к-во постов, опубликованных за предыдущий день.

		SELECT * FROM post WHERE date_created BETWEEN CURDATE() - INTERVAL 1 DAY AND CURDATE()
		
1.3. Вывести последние 5 опубликованных постов.

		SELECT * FROM `post` ORDER BY `date_created` DESC LIMIT 5
		
1.4. Вывести все посты со списком тегов, разделенных запятыми.	

	SELECT * FROM post p JOIN post_tag pt ON pt.post_id = p.id JOIN tag t ON t.id = pt.tag_id ORDER BY t.name
